# BearMap

  This project is from Berkeley's course CS61B(2019 spring). It builds a map similar to Google Maps, which supports the functions below.
<ol>
<li>Map rastering.</li>
<li>Autocomplete and Search place.</li>
<li>Navigation</li>
</ol>
